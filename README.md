# panviva-node-sdk-es6-sample

A quick demonstration of consuming [@panviva/node-sdk](https://www.npmjs.com/package/@panviva/node-sdk) inside an ES6 module.

## How to run
### Configuration
Fill the `.env` file.

```conf
PANVIVA_BASE=
INSTANCE_NAME=
API_KEY=
DOCUMENT_ID=
```
### Build
```bash
yarn install
```
### Run
```bash
yarn start
```
